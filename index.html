APIキーが必要です

アプリケーションを実行するには、Google Maps Platform APIキーが必要です。
プロジェクトのルートディレクトリに **`secrets.json`** ファイルを作成し、APIキーを記入してください。

{
                "Maps_API_KEY": "YOUR_ACTUAL_Maps_API_KEY"
            }

APIキーの取得方法と設定については、Google Maps Platform公式ドキュメントをご参照ください。
Maps JavaScript APIとGeocoding APIを有効にし、**HTTPリファラー制限を設定**してください。
